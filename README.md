# Tugas Akhir - Pemrograman Web 3

Deskripsi singkat

Proyek ini adalah tugas akhir praktikum Pemrograman Web 3 yang menampilkan contoh halaman web sederhana menggunakan HTML, CSS Tailwind Tujuan proyek adalah menerapkan konsep dasar pengembangan web front-end dan membuat halaman interaktif yang rapi.

Fitur

- Struktur halaman statis (HTML)
- Styling menggunakan CSS responsif sederhana

Teknologi

- HTML5
- CSS3
- JavaScript (vanilla)

Cara menjalankan

1. Buka folder proyek di file explorer.
2. Jalankan `index.html` dengan browser (klik dua kali atau buka lewat menu Open File di browser).

Struktur berkas (utama)

- `index.html` — halaman utama
- `css/` — (opsional) gaya
- `js/` — (opsional) skrip JavaScript
